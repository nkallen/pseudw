Intro

This game is designed to improve your Greek participle recognition, both in accuracy and in speed. It works like this: the computer presents an inflected participle, and you identify its tense, number, and so forth. You will be timed, so try hard not only to be correct but also fast. The computer will learn which types of forms you have trouble with and focus on them so as to improve your skills. For example, if you consistently mistake aorist forms, the computer will present those more often so you get more practice with them.

Tips

  • Try using the keyboard to answer. For example, type p to indicate Present tense and then hit Enter to submit your answer.

  • You can configure the game to select from a specific list of Greek verbs or only ask for specific parts of the inflection (e.g., only "tense").

  • You can share your game configurations with friends just by copying and pasting the URL from your browser's location bar. If the URL is too long, consider using a url shortener like bit.ly
